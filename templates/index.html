Welcome to the IN2000 API

Here's how you can use it:

Get All Activities

Send a GET request to /api/activities.

        "ActivityID": 1,

        "MinTemp": 10,
        "MinWind": 0
    }
]

Get Specific Activity

Send a GET request to /api/activities/{ActivityID} where {ActivityID} is the ID of the activity you want to retrieve.

{
    "ActivityID": 1,
    "ActivityName": "Running",
    "MaxRain": 2.5,
    "MaxTemp": 20,
    "MaxWind": 5,
    "MinRain": 0,
    "MinTemp": 10,
    "MinWind": 0
}

Update Activities Cache

Send a POST request to /db/updatecache with the X-API-KEY header.

{
    "message": "Cache updated successfully"
}
